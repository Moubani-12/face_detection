# Face Detection using Artificial Intelligence

## By: Moubani Das

## About This Project
A real-time face detection system built using Python and OpenCV.

## Features
- Face Detection (Blue box)
- Eye Detection (Green box)
- Smile Detection (Red box)
- Age Prediction
- Gender Prediction
- Auto Save detected faces

## Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier
- Caffe Deep Learning Models

## How to Run
1. Install required libraries:
   pip install opencv-python numpy
2. Download age_net.caffemodel and gender_net.caffemodel and place in project folder
3. Run the code:
   python face_detection.py

## Note
age_net.caffemodel and gender_net.caffemodel are not included due to GitHub file size limit (25MB).
